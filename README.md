# HighlightJS.swift

## What's this?

`HighlightJS.swift` is a code syntax highlight library using `Highlight.js`.

Checkout the sample project for more detail.

## Requirements

* iOS 8.0+
* macOS 10.11+
* tvOS 9.0+
* Xcode 8 with Swift 3

## Installation

#### CocoaPods

```ruby
pod 'HighlightJS'
```
